## Tелефонная книга.

Нужно сделать crud приложение добавлении номеров, так же сделать поиск по имени и номеру телефона. Написать тесты

##Технологии:

1. Язык Golang

2. PostgreSQL 

##Usage

$ docker-compose up postgres 

Build Service:

$ go build -o ./bin/service ./cmd/service/

Start Service:

$ ./bin/service

Test Service:

$ chmod +x ./test

$ ./test
